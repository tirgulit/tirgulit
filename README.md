## גישה לפאנל ניהול
- לחץ על `Ctrl + Shift + A` בכל מקום באתר
- הקש את הסיסמה: "1234"
- בפאנל הניהול תוכל לשנות:
  - צבעים ראשיים
  - כותרות
  - טקסטים

## הוספת תמונות
### הוספת לוגו חדש
1. הכן תמונת לוגו בפורמט PNG
2. גודל מומלץ: 400x400 פיקסלים
3. שמור את הקובץ בשם `logo.png`
4. העלה את הקובץ לתיקיית השורש של האתר

### הוספת תמונות נוספות
1. הכן תמונות באיכות טובה (מומלץ JPG או PNG)
2. גודל מומלץ: 1200x800 פיקסלים
3. העלה את התמונות לתיקיית השורש
4. עדכן את הנתיב בקוד בהתאם לשם הקובץ

## איורים
התיקייה `illustrations` מכילה את האיורים המשמשים באתר:
- `team.svg` - איור הצוות בעמוד הראשי
- נוספים בהמשך...

### עדכון איורים
1. שמור את האיור החדש בתיקיית `illustrations`
2. וודא שהצבעים תואמים את סכמת הצבעים של האתר:
   - ירוק: #4CAF50
   - כחול: #2196F3
   - כתום: #FF5722

## הוספת סרטון
### הוספת סרטון חדש
1. הכן סרטון בפורמט MP4
2. גודל מומלץ: 1080p
3. שמור את הקובץ בשם `video.mp4`
4. העלה את הקובץ לתיקיית השורש של האתר

### הוספת סרטון נוספות
1. הכן סרטונים באיכות טובה (מומלץ 1080p)
2. העלה את הסרטונים לתיקיית השורש
4. עדכן את הנתיב בקוד בהתאם לשם הקובץ
